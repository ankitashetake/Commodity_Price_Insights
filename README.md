**Business Objective:**

This project aims to analyze commodity pricing trends across Indian cities from 2019 to 2020 using structured SQL queries. The goal is to uncover high-impact market insights such as price variations, top-priced commodities, regional disparities, and availability patterns. These insights can aid policymakers, supply chain analysts, and retail strategists in making data-informed decisions related to agricultural and essential goods pricing and distribution.
